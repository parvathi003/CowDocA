# 🐄 CowDocAI

CowDocAI is an AI-powered chatbot for cattle disease detection. It combines image classification with Retrieval-Augmented Generation (RAG) to provide reliable information about cattle diseases.

## Features

- Image-based cattle disease detection
- RAG-based disease information retrieval
- AI chatbot for farmer assistance
- ChromaDB vector database
- EfficientNet-B0 image classification
- Verified veterinary knowledge base

## Tech Stack

- Python
- Streamlit
- ChromaDB
- Sentence Transformers
- OpenAI API
- EfficientNet-B0
- PyTorch
